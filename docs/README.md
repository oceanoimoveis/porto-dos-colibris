# Porto dos Colibris — Pagina de Plantas

Pagina interativa para o cliente escolher unidade do **Residencial Porto dos Colibris** (Fase 1).

## Como funciona
- Lista todas as unidades **DISPONIVEIS** dos Blocos 01, 05, 07 e 08.
- Filtros: Bloco, Coluna, Andar, Tipologia, Posicao Solar e Faixa de preco.
- Clique na planta para ampliar.
- Botao "Quero esta unidade" abre o WhatsApp com a unidade ja preenchida.

## Estrutura
- `index.html` — pagina principal
- `404.html` — pagina de erro
- `data/units.json` — base de dados das 35 unidades disponiveis
- `assets/brand/` — logos (Colibri + Oceano)
- `assets/tower/` — fachada e mapa de localizacao
- `assets/plants/` — 3 plantas tecnicas (40,75m2, 40,75m2 + quintal, 54,04m2)
- `assets/gallery/` — renders 3D do empreendimento
